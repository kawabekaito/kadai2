5421055 川邉開人

section element

(1) 文書に区間を作ることができる。divと同じような使い方が可能。特に意味はなし。見出しを中に置いておいた方が見やすくなる。

(2) 使用方法

<section>
 <h1>本日のニュース</h1>
 <p>みんな元気でした！</p>
</section>

(3) 実行結果

本日のニュース

みんな元気でした！


The h1, h2, h3, h4, h5, and h6 elements

(1) 見出しを作ることが可能。h1から順にh6まで見出しのサイズが小さくなっていく。

(2) 使用方法

<h1>川邉開人</h1>
<h2>川邉開人</h2>
<h3>川邉開人</h3>
<h4>川邉開人</h4>
<h5>川邉開人</h5>
<h6>川邉開人</h6>

(3) 実行結果

br element

(1) 改行したい部分に入れると改行することができる。<br>だけで使用可能、</br>はない。

(2) 使用方法

<p>こんにちは<br>今日はいい天気ですね。<br>ところで明日空いてますか？

(3) 実行結果

こんにちは
今日はいい天気ですね。
